# About Corner Break
Corner Break is a game where you have blocks surrounding a vault and you have to break the blocks in order to touch the vault and win the game.You have to control a ball by clicking to launch it and the ball's trajectory can be controlled by a platform.If the ball drops below the line of sight, you lose.There are a total of 3 levels in the game. The speed increases after each level

# How to play
you manuvar the ball by hitting it with a platform, the angle of its trajectory is decided by the place where the ball strikes the padel. Hitting at dead centre means it goes straight up at an angle of 90 degress and if you hit at an angle it gets projected at an angle as low as 30 degrees(for the paddle to the line of trajectory)




# Level changes
After each level a set changes take place to make the game more difficult
1. The speed of the ball increases, this makes it more harder to hit the ball with the moving platform
2. The number of hits it takes to break a block, this value is incremented each level and the number of hits required to break a block is displayed on the block itself and is updated in real timne
# Game win screen
 Once you complete each level you get a "VAULT CRACKED" screen displayiig that you won
 # Gameover screen
  The gameover screen is activated when the player loses displaying a "Game over" text this happens when the ball falls below the moving platform thus being unable to hit back. When this happens a square menu is displayed infront of the user which shows the gameover and clicking it restarts the game
